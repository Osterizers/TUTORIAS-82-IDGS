# 🚀 Sistema de Tutorías 82

> Proyecto integrador desarrollado con arquitectura Full Stack utilizando React, Node.js y MongoDB.

---

## 📌 Descripción

Sistema web para la gestión de tutorías académicas que permite administrar alumnos, tutores y sesiones de manera eficiente.

---

## 🧠 Arquitectura del sistema

El proyecto está dividido en:

- 🔙 Backend API (Node.js + Express)
- 🎨 Frontend (React)
- 🗄️ Base de datos (MongoDB)

---

## 🛠️ Tecnologías utilizadas

### Backend
- Node.js
- Express

### Frontend
- React
- JavaScript (ES6+)
- Bootstrap / CSS

### Base de Datos
- MongoDB
